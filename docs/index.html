PetSave

Um App para ajudar na Doação de Sangue para Pets

Sobre o Projeto

O PetSave é uma aplicação dedicada a facilitar doações de sangue para animais de estimação, conectando tutores de pets que precisam de doações com potenciais doadores, facilitando o encontro de ambos com maior rapidez e aumentando as chances de salvar vidas de outros pets mais rapidamente.

Funcionalidades

  • Cadastro de tutores
  • Cadastro de pets
  • Informações gerais sobre os pets e tutores

Como Funciona

1. Cadastre-se no aplicativo

2. Registre seu pet

3. Encontre doadores ou ajude seu pet a salvar uma vida!
